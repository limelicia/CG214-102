The roTopo Game

About roTopo:

roTopo is a 2D puzzle game on the web browser with 3D rotation. The objective of the game is for the player to clear the puzzles by stepping on every tile once. Player controls avatar with keyboard, or by swiping with finger on mobile devices. With a constantly moving character, the game requires the player to be quick-witted and nimble.
